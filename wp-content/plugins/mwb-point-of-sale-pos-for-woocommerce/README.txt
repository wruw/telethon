Incorporating the plugin, the admin can generate barcodes for different products with ease making use of the single barcode generator button.

Your clients may now place POS purchases with wallet-based transactions and generate precise and comprehensive billing invoices based on their orders using the plugin capabilities.

**11)** **Outstanding Product Search**
Outlet/Store managers can easily search products over the WooCommerce POS dashboard using the outstanding product search feature. Search covers almost all products listed on your WooCommerce store.

**12)** **Offers Effortless Inventory Management**
Offers efficient inventory management that is seamless and does not require you to manually update order data. With a Point of Sale for WooCommerce plugin, you can track the sales of a product, and you can also know when it’s time to upgrade your stock.

== Installation ==

= Automatic installation =
Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of the plugin, log in to your WordPress dashboard, navigate to the Plugins menu, and click Add New.

In the search field type **"MWB Point of Sale (POS) for WooCommerce"** and click Search Plugins. Once you've found our MWB Point of Sale (POS) for WooCommerce plugin you can view details about it such as the point release, rating, and description. Most importantly, of course, you can install it by simply clicking **"Install Now"**.

= Manual installation =
The manual installation method involves downloading our MWB Point of Sale (POS) for WooCommerce and uploading it to your webserver via your favorite FTP application. The WordPress codex contains [**instructions on how to do this here**](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

= Updating =
Automatic updates should work like a charm; as always though, ensure you backup your site just in case.

== Frequently Asked Questions ==


= Do sellers have to manually apply tax to the cart total? =
Yes, sellers should therefore input tax into the POS system.
 
= We offer the customer multiple payment options at the POS counter? =
Yes, for the time being, you can accept cash and cards as payment options.

= In my store, I have several POS counters. Is it mandatory for someone like me to create admin user roles for all the POS counters? =

For the time being, billings on counters can be accessed from a single admin login across multiple screens.

= Can we add a Logo with the Point of Sale plugin? =
 Yes, you can add a logo with the MWB-Point of Sale plugin 

= Can we customize our Point of Sale Home page? =
Yes, you can customize your POS Homepage with our Point of Sale plugin.

= Can we add a seller or manager profile with POS for a WooCommerce plugin? =
  Yes, you can update all profiles with the Point of Sale for WooCommerce plugin.

 = Can we get a notification with a POS plugin? =
 You can easily set the desktop as well as email notifications for orders/bills generated using the POS system.

= Does the admin have to create each product barcode manually? =
Admin can easily create barcodes of all the listed products in WooCommerce through one click only
