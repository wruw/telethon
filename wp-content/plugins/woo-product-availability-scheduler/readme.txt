== Description ==

This Plugin you can manage the availability of all your woocommerce products, scheduling it throughout the day of the week
You will have the flexibility to choose Date and time range when your products will be available to your customers for purchasing, automatically hiding/showing the “add to cart button” on shop, category and product pages, showing a countdown timer. 

After enable status of Schedule Availability for any product then the product will still be present in your WooCommerce shop before the start date, but a visitor will not be able to buy it, because the "Add-to-Cart" button will not be present. You can display a custom message to the visitor,

== COUNTDOWN TIMERS ==

Woocommerce Product Availability Scheduler plugin gives you the option to display countdown timers that lets your customers know how much time misses to start purchasing time, to the end purchasing time or to the expiration date and time. 

== EXPIRING DATE TIME ==

This Plugin you can set an expiring date time for every product. After this date time the product will be automatically hidden and unavailable for purchasing . Optionally the product can be still setted as visbile.

Also works with WordPress Multisite installs (each blog from the network has it’s own maintenance settings).

<code><?php echo do_shortcode('[wpas_schedule_product]'); ?> </code>

**Plugin Features**

* Manage product Availability
* Hide Add-to-Cart Button when shop is closed
* Display next opening time
* Purchase availability per time range, per product
* Manually disable, Enable schedule Product availability
* Countdown Timer for pre schedule Purchase Remaining time 
* Supported multisite
 


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

After Plugin Active go to Products-> Add new -> Availability Scheduler(tab).
